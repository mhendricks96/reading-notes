Selectors

selectors indicate which element the rule will apply to.

Declarations

Declarations indicate how the element referred to in the slector should be styled. the declaration is split into 2 parts.

Property

The first part of the declararation is the property. this is what indicates what about the eement will be changed.

Value

the second part of the declararation is the Value. This tells us what exactly the property will be set to.

Where is the CSS located?

Internal

Internal css is located within the actual html

External

External css is located on a seperate file from the html. It is connected by a link element in the head of the html. This is helpful because we can link multiple pages of the website to the same css
